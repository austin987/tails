Several security holes affect Tails 1.0.1.

We strongly encourage you to upgrade to Tails 1.1 as soon as possible.

Details

  • libxml2: Debian Security Advisory 2978
  • dbus: Debian Security Advisory 2971
  • linux: Debian Security Advisory 2972
  • gnupg: Debian Security Advisory 2967
  • tiff: Debian Security Advisory 2965
  • apt: Debian Security Advisory 2958
