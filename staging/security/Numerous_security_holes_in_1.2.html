Several security holes that affect Tails 1.2 are now fixed in Tails 1.2.1.

We strongly encourage you to upgrade to Tails 1.2.1 as soon as possible.

Details

  • Tor Browser and its bundled NSS: Mozilla Foundation Security Advisory 2014-83, Mozilla Foundation Security Advisory 2014-85, Mozilla Foundation Security Advisory 2014-87, Mozilla Foundation Security Advisory 2014-88, Mozilla Foundation Security Advisory 2014-89 and Mozilla Foundation Security Advisory 2014-90
  • wpa: Debian Security Advisory 3052 (CVE-2014-3686)
  • openssl: Debian Security Advisory 3053 (CVE-2014-3513, CVE-2014-3566, CVE-2014-3567, CVE-2014-3568)
  • pidgin: Debian Security Advisory 3055 (CVE-2014-3694, CVE-2014-3695, CVE-2014-3696, CVE-2014-3698)
  • libtasn1-3: Debian Security Advisory 3056 (CVE-2014-3467, CVE-2014-3468, CVE-2014-3469)
  • libxml2: Debian Security Advisory 3057 (CVE-2014-3660)
  • linux: Debian Security Advisory 3060 (CVE-2014-3610, CVE-2014-3611, CVE-2014-3645, CVE-2014-3646, CVE-2014-3647, CVE-2014-3673, CVE-2014-3687, CVE-2014-3688, CVE-2014-3690, CVE-2014-7207)
  • wget: Debian Security Advisory 3062 (CVE-2014-4877)
  • curl: Debian Security Advisory 3069 (CVE-2014-3707)
  • nss: Debian Security Advisory 3071 (CVE-2014-1544)
  • file: Debian Security Advisory 3072 (CVE-2014-3710)
  • libgcrypt11: Debian Security Advisory 3073 (CVE-2014-5270)
  • ppp: Debian Security Advisory 3079 (CVE-2014-3158)
  • openjdk-7: Debian Security Advisory 3080 (CVE-2014-6457, CVE-2014-6502, CVE-2014-6504, CVE-2014-6506, CVE-2014-6511, CVE-2014-6512, CVE-2014-6517, CVE-2014-6519, CVE-2014-6531, CVE-2014-6558)
  • flac: Debian Security Advisory 3082 (CVE-2014-8962, CVE-2014-9028)
  • mutt: Debian Security Advisory 3083 (CVE-2014-9116)
