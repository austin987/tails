Tails, The Amnesic Incognito Live System, version 0.20.1, is out.

All users must upgrade as soon as possible: this release fixes numerous security issues.

  1. Changes
  2. Known issues
  3. I want to try it or to upgrade!
  4. What's coming up?

Changes

Notable user-visible changes include:

  • New features

    • Install Tor 0.2.4.17-rc.
    • Install Iceweasel 17.0.9esr with Torbrowser patches.
    • Install Linux kernel 3.10-3 (version 3.10.11-1).

  • Bugfixes

    • Remount persistence devices read-only at shutdown/reboot time (#6228).
    • Greeter: display a warning icon on admin password mismatch and on persistence unlocking failure. Thanks to Andres Gomez Ramirez andres.gomez@cern.ch for the fix!
    • Fix the I2P IRC account.
    • Allow installing "Priority: standard" packages that we do not install by default.

  • Minor improvements

    • Additional software: point to the system log on upgrade failure.

  • Localization

    • Many translation updates all over the place.

See the online Changelog for technical details.

Known issues

No new known issue but longstanding known issues.

For users of the Pidgin persistence feature

If you have turned on the persistent Pidgin configuration feature, then you have to update the following configuration in Pidgin:

Tools ▸ Preferences ▸ Proxy

  1. Activate Use remote DNS with SOCKS4 proxies.
  2. Choose SOCKS5 as the Proxy type.
  3. Type 127.0.0.1 as the Host.
  4. Type 9050 as the Port.

Click on the Close button.
