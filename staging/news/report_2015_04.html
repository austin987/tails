This report is a bit special: since we release every 6 weeks, there is no release this month. The next one is planned for the 12th of May. Therefore there are no code news, the work being done will be reported when it's released :)

However, like any Free Software project, Tails is not only about the code, so here are the news about the other parts of the project!

  1. Documentation and website
  2. User experience
  3. Infrastructure
  4. Outreach
  5. On-going discussions
  6. Translation and internationalization
  7. Metrics

Documentation and website

  • We wrote a small documentation about Claws Mail.

  • We corrected some details in the manual installation documentations, the verification documentation, and related FAQ questions.

  • We added a FAQ entry about why we ship the GNOME Desktop.

  • We added a FAQ entry about why we don't sell preinstalled Tails devices.

  • We added (yet another) FAQ entry about why you shouldn't edit your torrc.

  • ... and various small improvements to the documentation all over the place :)

User experience

  • On the ISO verification extension:

    • We started a security discussion about the possible threats from inside the browser. We need your expertise!
    • We finished a detailed wireframe of the various screens of the extension.

  • On the web assistant for getting started with Tails:

    • We finished a prototype for the 3rd iteration of the router (the set of introductory questions) and tested it with several people.
    • We started a discussion about several terminology issues.
    • We proposed a URL scheme, a JavaScript policy, and a compatibility list for Mac.

  • We published some guidelines for user testing.

Infrastructure

  • Our test suite now covers 177 scenarios, 15 more than in March.

  • We talked about having an underlay for UX or blueprints

  • The automated builds specification discussion reached a conclusion

  • We updated the APT suites handling

Outreach

  • intrigeri talked about contributing to Tails via Debian at a MiniDebConf in Lyon, France.

Upcoming events

  • A talk about Tails will take place at a MiniDebConf in Bucharest, Romania May 16th.

  • People are organizing a workshop about Tails in Brest, France in June.

On-going discussions

  • The discussion continued regarding GUI for encrypted volumes from LUKS/TrueCrypt container files

  • A discussion has started concerning the migration from Python 2 to Python 3

  • People talked about Tails-like system for tablets. Hint: that's not coming soon.

  • Someone had a proposal concerning a Group Installer
