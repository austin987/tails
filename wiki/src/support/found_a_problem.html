[[!meta title="Found a problem?"]]

So you have a problem with Tails. Well, let's see how to get it fixed.

To begin with, make sure you are using the latest version, as [[upgrading|doc/upgrade]] might solve your problem. If it does not, please read on.

Check if the problem is already known. You can have a look at:

  • the list of known issues
  • the list of things to do
  • the list of things that will be in the next release
  • the list of known bugs
  • the list of bugs that will fixed in the next release

Found a bug?

If you've found a bug in Tails, please follow the [[bug reporting guidelines|doc/first_steps/bug_reporting]].

Need a new feature?

If you would like to see a new feature in Tails, [[!tails_website wishlist desc="tell us about it!"]]
